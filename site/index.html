Фреймворк для алгоритмической торговли на финансовых рынках

Инструментарий для работы с историческими данными рынка, проектирования, тестирования и оптимизации торговых стратегий для автоматизированной торговли на российских биржах.

Анализ исторических данных

Инструменты для загрузки, обработки и анализа исторических рыночных данных с российских бирж. Возможность визуализации и выявления паттернов.

Разработка стратегий

Создание и оптимизация торговых стратегий на основе технического анализа, статистических методов и математических моделей.

Автоматизированная торговля

Реализация автоматического исполнения торговых стратегий с контролем рисков и мониторингом результатов.

О проекте

Архитектура фреймворка

Наш фреймворк представляет собой модульную систему, состоящую из нескольких взаимосвязанных компонентов, каждый из которых отвечает за определенный аспект алгоритмической торговли.

Основные модули включают в себя систему сбора и хранения данных, инструменты для анализа и визуализации, модуль разработки стратегий, бэктестер для проверки эффективности стратегий и модуль исполнения для реальной торговли.

Все компоненты взаимодействуют через единый API, что обеспечивает гибкость и расширяемость системы.

Наша команда

Наш проект объединяет 23 студента Московского политехнического университета, разделенных на три специализированные команды, каждая из которых отвечает за ключевые аспекты разработки фреймворка для алгоритмической торговли.

23

Команда разработки стратегий

8 человек, специализирующихся на создании и оптимизации торговых алгоритмов.

  • Разработка индикаторов технического анализа
  • Создание логики торговых стратегий
  • Оптимизация параметров алгоритмов
  • Анализ эффективности стратегий

Команда тестирования

8 человек, отвечающих за проверку и валидацию разработанных стратегий.

  • Создание системы бэктестирования
  • Разработка метрик эффективности
  • Тестирование на исторических данных
  • Визуализация результатов

Команда базы данных

7 человек, занимающихся сбором и управлением данными.

  • Интеграция с API бирж
  • Проектирование структуры БД
  • Оптимизация хранения и доступа
  • Обработка и очистка данных

Журнал проекта

Успешно пройдена защита проекта

26 мая 2025

Команда проекта успешно представила и защитила финальную версию фреймворка для алгоритмической торговли. Все поставленные цели были достигнуты, а функциональность системы полностью соответствует требованиям.

В ходе защиты были продемонстрированы все ключевые компоненты системы, включая модули сбора и анализа данных, разработки и тестирования торговых стратегий.

Успешно пройден 2 промежуточный отчет (аттестация)

14 мая 2025

Вторая аттестация проекта успешно пройдена. Представлены результаты разработки системы тестирования торговых стратегий и модуля оптимизации параметров.

Комиссия отметила высокое качество реализации системы бэктестирования и эффективность алгоритмов оптимизации торговых стратегий.

Получены рекомендации по дальнейшему развитию проекта и подготовке к финальной защите.

Успешно пройден 1 промежуточный отчет (аттестация)

19 марта 2025

Первая аттестация проекта успешно пройдена. Представлена базовая архитектура системы и результаты разработки модуля сбора и хранения данных.

Комиссия положительно оценила выбранные технические решения и качество реализации базовых компонентов системы.

Определены направления дальнейшей работы и согласован план развития проекта.
